# vue-ts

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
# 本地开发地址
npm run local
# 开发环境地址
npm run dev
# 测试开发地址
npm run test
# 生产开发地址
npm run prod
```

### Compiles and minifies for production
```
# 开发打包
npm run build-dev
# 测试打包
npm run build-test
# 生产打包
npm run build-prod
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
